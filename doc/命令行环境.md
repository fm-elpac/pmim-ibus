## 环境变量

- `PMIMS_PORT`

  pmim-server 监听的 HTTP 端口号.

  `http://127.0.0.1:{PMIMS_PORT}`

- `XDG_RUNTIME_DIR`

  pmim-server 口令文件: `{XDG_RUNTIME_DIR}/pmim/server_token`

- `PMIM_DEBUG`

  开启调试功能.

  - `PMIM_DEBUG=1`

    打开候选框窗口开发者工具.

    ```js
    窗口.im1.webContents.openDevTools();
    ```
